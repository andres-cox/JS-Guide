ADD / REMOVE Items

items = [12, "apple", "orange", 23, 53];

Adds items to the end
items.push("strawberry"); //returns new total length
items = [12, "apple", "orange", 23, 53, "strawberry"];

Extracts an item from the end
items.pop(); //returns last element
items = [12, "apple", "orange", 23];

Extracts an item from the beginning
items.shift(); //returns first element
items = ["apple", "orange", 23, 53];

Adds items to the beginning
items.unshift(45); //returns new total length
items = [45, 12, "apple", "orange", 23, 53];

Returns a copied slice from the original array
items.slice() //returns a copy from the original array

Changes the content of the array in place
items.splice(1, 1, "banana")) // returns elements removed
items.splice(index, #elements to remove, elements to add)) // returns elements removed
items = [12, "banana", "orange", 23, 53];

Merge two or more arrays
items2 = [12, "potate", "tomate", 79, 94]
items.concat(items2); //returns new array merged
